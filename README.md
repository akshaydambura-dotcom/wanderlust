Wanderlust 🌍

A full-stack travel listing web application where users can explore, create, and manage travel destination listings.

🚀 Live Demo

https://wanderlust-xyh0.onrender.com/listings

📌 Features

Create, edit, and delete travel listings

View listings with images and descriptions

RESTful API architecture for backend operations

Responsive UI for smooth browsing experience

Cloud deployment for global accessibility

Secure routing and middleware handling

git clone https://github.com/akshaydambura-dotcom/wanderlust.git

Move to project folder

cd wanderlust

Install dependencies

npm install

Run the server

node app.js

Open in browser

http://localhost:3000
